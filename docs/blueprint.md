# **App Name**: TaxClarity NG

## Core Features:

- Tax Comparison: Compare income tax before and after 2026 based on user-provided income details.
- Income Type Sensitivity: Dynamically adjust calculations based on the user's income type (salary, business, mixed).
- Cash Income Adjustment: Incorporate a user-adjustable percentage to account for estimated cash income.
- Result Summary: Clearly present absolute and percentage change in tax liability, with increase/decrease indicators.
- Plain-English Explanation: Use generative AI tool to produce a short, plain-English explanation of the tax law changes driving the results, personalized to the user's income details and the specific calculation results.
- Report Generation Hook: Offer a link to generate a more detailed report (even if the MVP report is basic).

## Style Guidelines:

- Background color: Off-white (#F9F9F9) to provide a calm and clean terminal-like feel.
- Primary color: Black (#000000) for the main text to ensure high readability and contrast.
- Accent color: Soft Green (#A7D1AB) inspired by the Nigerian flag, used for interactive elements and highlights.
- Body and headline font: 'Inter', a sans-serif font known for its clean, modern, and readable design. This will maintain a terminal inspired look.
- Single-column layout with generous spacing to ensure clarity and focus; emulating a streamlined terminal interface.
- Minimal use of icons. Use simple, geometric shapes if needed to complement terminal aesthetic.
- Subtle transitions and loading animations (e.g., typing cursor) to provide feedback without disrupting the user flow.
